despre ce se vorbeste pe scurt in acest tutorial?
În această secțiune, vorbesc despre cum să creați un prompt în aplicația dvs. realizată cu Python și biblioteca Streamlit (st).

Promptul este o formă de interacțiune între utilizator și aplicație, unde utilizatorul poate introduce o întrebare sau cerință. 
În acest caz, promptul este un camp de text numit "Ask any question about the webpage" care permite utilizatorului să introducă 
o întrebare despre pagina web încărcată.
